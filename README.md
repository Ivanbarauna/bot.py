# Bot de Discord – Armazenamento de Links e Imagens

Este projeto consiste em um **bot de Discord simples**, desenvolvido com o objetivo de **armazenar links e imagens** de forma organizada. Ele foi criado como um **projeto de estudo**, marcando meus primeiros passos após iniciar o curso de **Análise e Desenvolvimento de Sistemas**.

A proposta principal foi aplicar conceitos básicos de programação, integração com APIs e boas práticas iniciais de desenvolvimento.


## Funcionalidades

* Armazenamento de links (ex.: jogos, sites, referências)
* Envio de mensagens utilizando **embeds**
* Organização simples e prática do conteúdo
* Integração direta com o Discord


## Requisitos para Executar o Bot

1. **Criar uma aplicação no Discord Developer Portal**

   * É necessário ter sua própria aplicação/bot registrada no Discord.

2. **Configurar o Token com Segurança**

   * Após criar o bot, obtenha o *token* da aplicação.
   * **Por segurança, nunca exponha o token no código-fonte.**
   * Armazene-o em um arquivo `.env`.

3. **Configurar Embeds e Links**

   * Defina os links e imagens que o bot irá armazenar e exibir.
   * Personalize as embeds de acordo com sua necessidade.


## Tecnologias Utilizadas

* Discord API
* Bot para Discord
* Variáveis de ambiente (`.env`)
* Conceitos iniciais de Análise e Desenvolvimento de Sistemas


## Objetivo do Projeto

Este projeto tem caráter **educacional**, servindo como prática para:

* Entender o funcionamento de bots no Discord
* Trabalhar com tokens e variáveis de ambiente
* Aplicar organização básica de código
* Desenvolver lógica e estrutura de um projeto real
